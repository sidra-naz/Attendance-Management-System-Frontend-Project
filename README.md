# ** Attendance Management System**

**Attendance Management System** is a responsive web application built using **HTML5**, **CSS3**, and **JavaScript**. It helps manage student records by allowing users to add, edit, delete, search, and track attendance efficiently through a simple and user-friendly interface.

## ** Features**

* Add new student records
* Edit existing student information
* Delete student records
* Search students by name
* Mark students as **Present** or **Absent**
* Color indicators for attendance status
* Responsive and user-friendly interface

## **Technologies Used**

* HTML5
* CSS3
* JavaScript (ES6)

## **Project Purpose**

The purpose of this project is to practice front-end web development by building a simple student attendance management system. It demonstrates the use of JavaScript for DOM manipulation, CRUD (Create, Read, Update, Delete) operations, search functionality, and dynamic data handling while providing a clean and responsive user interface.
